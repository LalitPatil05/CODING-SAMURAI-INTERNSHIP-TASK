# Simple Calculator (Java)

## 📌 Description
This is a basic calculator developed in Java.
It performs:
- Addition
- Subtraction
- Multiplication
- Division

## 🛠 Skills Used
- Java Basics
- Scanner (User Input)
- Conditional Statements
- Switch Case
- Loops

## ▶️ How to Run
1. Compile:
   javac Calculator.java

2. Run:
   java Calculator
